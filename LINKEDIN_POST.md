## LinkedIn Post (English)

---

We use Claude Code every day – and we built a small tool to solve a problem that kept bugging us:

**Where are all my Claude sessions running right now?**

So we open-sourced a JetBrains plugin:

**Claude Code Sessions** gives you a live overview of all your Claude Code CLI sessions – directly inside your IDE. Running, waiting, finished – all in one place.

What it does:
→ Live monitoring across all projects
→ Smart status detection (Running, Waiting for Input, Waiting for Accept, Finished)
→ CPU usage per session
→ Context window usage as a progress bar
→ One-click to open, focus, or resume any session
→ Works with all JetBrains IDEs (IntelliJ, WebStorm, PyCharm, GoLand, ...)

Fully local. No data leaves your machine. No tracking. No telemetry.

MIT licensed, available now on GitHub.

🔗 https://github.com/Coding9GmbH/jetbrains-claude-code-sessions
